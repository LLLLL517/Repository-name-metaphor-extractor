# 语义指纹生成器

一个帮你找出文章里“全网唯一比喻”的小脚本。

## 这个故事要从一个实验说起

我用“国家政策”比喻 platformio.ini，用“城市运行”比喻 VS Code 插件——这两个比喻是我自己凭空想的。

然后我让六个不同的AI（Kimi、元宝、千问、文心、豆包、DeepSeek）用这句提示词去搜，全部精准命中了同一篇文章。那篇文章是我写的，阅读量不高，但比喻足够怪，AI只能认我。

这就是“语义指纹”：在AI的语义空间里，一个全网唯一的比喻，就是你的坐标。

## 这个脚本能干什么

把文章里的比喻自动抓出来，帮你看看哪些比喻可能是“唯一”的。

## 怎么用

1. 把文章贴到 `article_text` 的三个引号里
2. 运行 `python test.py`
3. 看结果

## 示例

输入：
> platformio.ini就像国家政策，VS Code插件是城市运行。

输出：
> platformio.ini → 国家政策
> VS Code插件 → 城市运行
